Va dans terminal :
- Va dans bon répertoire
- docker-compose build && docker-compose up -d
	# construira nos images
	# démarrera les conteneurs comme déclaré dans notre fichier docker-compose
	# Au démarrage, notre backend conteneur s'exécutera python main.py 
	# à partir de notre $WORKDIR, où nous montons notre /backend/répertoire local.

Va dans frontend :
- npm run start
	# ouvrira l'application React dans votre navigateur 
	# par défaut à l' adresse http://localhost:3000
